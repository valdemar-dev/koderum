# Koderum
Keyboard based fully graphical text editor.  
  
## Libraries / Bindings  
- OpenGL Odin Bindings  
- GLFW Odin Bindings  
- FreeType2 Odin Bindings  
- Tree-Sitter Odin Bindings  
  
  
## Installation  
Head to the releases page in github, grab the latest build, and follow the instructions.
